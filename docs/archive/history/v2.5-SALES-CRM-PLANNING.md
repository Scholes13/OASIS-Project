## 📋 Overview

Version 2.5 introduces **Sales CRM Module** dengan 2 core features:
1. **Daily Activities** - Sales activity logging (calls, visits, meetings, blitz)
2. **Contacts** - Customer/lead contact management with company tracking

**Key Requirement**: Sales role tetap punya akses **Purchase Request** + tambahan **CRM Module**

---

## 🎯 Business Requirements

### Sales Role Capabilities
Sales team membutuhkan:
- ✅ **PR Access** (Create, Edit, View own PRs)
- ✅ **Daily Activity Logging** (Track sales activities)
- ✅ **Contact Management** (Manage leads/customers from activities)
- ✅ **Company Visit History** (Track last visit per company/department)
- ✅ **Analytics Dashboard** (Activity stats, top contacts, etc)

### Permission-Based Access
- **Super Admin**: Full access (all modules + settings)
- **Admin**: Full access (all modules, no system settings)
- **Department Head**: Approve PRs + view team data
- **Sales** ← NEW: PR access + CRM access (own data only)
- **User**: PR access only (basic)

## 🗄️ Database Schema

### 1. Activities Table

```php
Schema::create('activities', function (Blueprint $table) {
    $table->id();
    $table->foreignId('business_unit_id')->constrained();
    $table->foreignId('user_id')->constrained(); // Sales person
    $table->foreignId('contact_id')->nullable()->constrained(); // Linked contact
    
    // Activity Details
    $table->date('activity_date');
    $table->enum('activity_type', [
        'call',
        'visit', 
        'meeting',
        'blitz',
        'follow_up',
        'other'
    ]);
    $table->string('title');
    $table->text('description')->nullable();
    $table->string('location')->nullable();
    
    // Time Tracking
    $table->time('start_time')->nullable();
    $table->time('end_time')->nullable();
    $table->integer('duration_minutes')->nullable();
    
    // Results
    $table->enum('result', [
        'success',
        'follow_up_needed',
        'no_answer',
        'rejected',
        'other'
    ])->nullable();
    $table->text('notes')->nullable();
    
    // Status
    $table->enum('status', [
        'planned',
        'completed',
        'cancelled'
    ])->default('planned');
    
    $table->timestamps();
    $table->softDeletes();
    
    // Performance Indexes (Task 1.1 Standard)
    $table->index(['user_id', 'status', 'activity_date']);
    $table->index(['business_unit_id', 'status', 'activity_date']);
    $table->index(['contact_id', 'activity_date']);
    $table->index(['activity_type', 'status']);
    $table->index(['activity_date', 'status']);
});
```

### 2. Contacts Table

```php
Schema::create('contacts', function (Blueprint $table) {
    $table->id();
    $table->foreignId('business_unit_id')->constrained();
    $table->foreignId('created_by')->constrained('users');
    $table->foreignId('assigned_to')->nullable()->constrained('users'); // Owner
    
    // Auto-generated Code
    $table->string('code')->unique(); // CONT-WNS-00001
    
    // Personal Info
    $table->string('name');
    $table->string('email')->nullable();
    $table->string('phone')->nullable();
    $table->string('mobile')->nullable();
    $table->date('birth_date')->nullable();
    
    // Professional Info
    $table->string('company')->nullable();
    $table->string('department')->nullable();
    $table->string('position')->nullable(); // Jabatan
    
    // Social Media (JSON)
    $table->json('social_media')->nullable();
    // Example: {"linkedin": "url", "instagram": "@username"}
    
    // Contact Source
    $table->enum('source', [
        'blitz',
        'call',
        'referral',
        'website',
        'event',
        'other'
    ])->default('other');
    $table->foreignId('source_activity_id')->nullable()->constrained('activities');
    
    // Status & Category
    $table->enum('status', [
        'active',
        'inactive',
        'archived'
    ])->default('active');
    $table->enum('category', [
        'lead',
        'prospect',
        'customer',
        'partner'
    ])->default('lead');
    
    // Additional Data
    $table->text('address')->nullable();
    $table->text('notes')->nullable();
    
    $table->timestamps();
    $table->softDeletes();
    
    // Performance Indexes (Task 1.1 Standard)
    $table->index(['business_unit_id', 'status', 'created_at']);
    $table->index(['assigned_to', 'status']);
    $table->index(['company', 'status']);
    $table->index(['category', 'status']);
    $table->index('code');
});
```

### 3. Company Visit History Table

```php
Schema::create('company_visit_history', function (Blueprint $table) {
```

## 🎨 Key Features

### Feature 1: Create Activity → Auto Create Contact

**User Flow**:
1. Sales membuat activity (call/visit/blitz)
2. Saat activity, bertemu orang baru
3. Input data orang (nama, jabatan, company, dll)
4. Sistem auto-create Contact dari activity
5. Link contact ke activity
6. Update company visit history

**Implementation**:
```php
// ActivityService.php
public function createActivityWithContact(
    array $activityData, 
    ?array $contactData = null
): Activity {
    DB::beginTransaction();
    
    try {
        // 1. Create Activity
        $activity = Activity::create($activityData);
        
        // 2. If contact provided, create or update
        if ($contactData) {
            $contact = Contact::updateOrCreate(
                ['phone' => $contactData['phone']],
                $contactData
            );
            
            // 3. Link to activity
            $activity->update(['contact_id' => $contact->id]);
            
            // 4. Update visit history
            $this->updateCompanyVisitHistory($contact, $activity);
        }
        
        DB::commit();
        return $activity;
        
    } catch (\Exception $e) {
        DB::rollBack();
        throw $e;
    }
}
```

### Feature 2: Contact Profile with Activity History

**Display**:
- Personal info (nama, email, phone, birth date)
- Professional info (company, department, position)
- Social media links (LinkedIn, Instagram, Facebook)
- Activity history (all interactions with this person)
- Company visit summary (last visit, total visits)

**Livewire Component**:
```php
// Contact/Show.php
use App\Livewire\Traits\HasLazyLoading; // Reusable!

class Show extends Component
{
    use HasLazyLoading;
    
    public Contact $contact;
    
    #[Computed]
    public function activities()
    {
        return $this->contact->activities()
            ->with('user')
            ->latest('activity_date')
            ->paginate(10);
    }
    
    #[Computed]
    public function companyHistory()
    {
        return $this->contact->companyVisitHistory;
    }
}
```

- Total visits per company
