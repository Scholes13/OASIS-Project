# Cashflow Projection PRD (Latest)

Dokumen aktif untuk PRD Cashflow Projection ada di:

- `docs/plans/2026-02-18-cashflow-projection-prd.md`

## Keputusan Kunci Saat Ini

- Scope MVP: per Business Unit, horizon 12 bulan, cycle `draft/published`.
- Template spesial: `ACC`, `HR`, `CFC`; selain itu masuk `Other Dept` berdasarkan departemen aktif di BU terpilih.
- Threshold warning saldo minimum: global `200.000.000` untuk semua BU.
- Mapping mengikuti baseline Excel `IN/OUT-DEPT-ACTION`.
- Untuk WNS, `Other Dept` = dept aktif selain `ACC`, `HR`, `CFC`.
- `Takshaka` adalah BU tersendiri (`TEE`), bukan dept WNS.
- `Reservasi` pada baseline legacy diarahkan ke `Corporate Travel (CT)` di BU `MRP`.
- Visibilitas sidebar modul hanya untuk `Head Department` dan `Finance/CFC`; user lain tidak melihat menu dan tidak bisa akses URL modul.
